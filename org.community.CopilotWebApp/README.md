# Copilot WebApp

Copilot WebApp is a community-maintained wrapper that runs [Microsoft Copilot](https://copilot.microsoft.com) in Chromium app mode, providing a dedicated desktop experience.

## Features
- Standalone window for Copilot
- Integrates with your Linux desktop
- Easy installation via Flathub

## Disclaimer
This is an **unofficial** package. It is not affiliated with, endorsed, or supported by Microsoft.  
It simply provides a convenience wrapper around the Copilot web interface.
